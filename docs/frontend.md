# Frontend Documentation

## Overview

The frontend is developed using React, TypeScript, and Vite.

The frontend provides the user interface for inventory management, point of sale transactions, sales record viewing, and dashboard monitoring.

The frontend communicates with the backend through REST API requests.

---

# Technologies Used

* React
* TypeScript
* Vite
* HTML
* CSS
* JavaScript

---

# Pages

## Dashboard

The dashboard displays system information required by the assessment.

Displays:

* Total number of products
* Current inventory levels
* Total sales transactions
* Recent sales activity

---

## Products Page

The Products page handles inventory management.

Features:

* Add new products
* Edit existing products
* Delete products
* View product list
* Display product price
* Display stock quantity
* Display stock availability

---

## POS Page

The POS page handles sales transactions.

Features:

* View available products
* Add products to shopping cart
* Update item quantities
* Remove items from cart
* Automatically calculate totals
* Complete sales transactions

---

## Sales Transactions Page

The Sales Transactions page displays stored transaction records.

Displays:

* Receipt number
* Transaction date
* Total amount
* Transaction details

---

# API Communication

The frontend communicates with the backend through HTTP requests.

Operations used:

* GET
* POST
* PUT
* DELETE

---

# Frontend Responsibilities

The frontend is responsible for:

* Displaying product data
* Managing shopping cart interactions
* Displaying dashboard information
* Sending transaction requests
* Displaying sales records
* Presenting data through a user-friendly interface

# User Interface Requirements

The frontend follows the assessment requirements:
